# Frontend Development Instructions

## Technology Stack
- Next.js 14+
- React
- TypeScript
- Better Auth Client
- Tailwind CSS

## Key Principles
1. All API calls include authentication tokens
2. Handle authentication state properly
3. User-friendly error messages
4. Responsive design
5. Type-safe API calls

## File Structure
- `app/` - Next.js app directory
- `components/` - React components
- `lib/` - Utility functions and API clients

## Development
```bash
npm run dev
```
